The **Igniter** can be used to light a fire in front of itself, when given a redstone signal.

It has 3 modes, set in its GUI:
- Toggle: Lights when powered, extinguishes when unpowered
- Keep Ignited: Keeps the fire lit while powered, does nothing when unpowered
- Ignite: Lights when powered, does nothing when unpowered
